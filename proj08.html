Mythic Makeup

This project was developed in the context of the curricular unit of Multimedia Laboratory I of the NTC degree.

Development tools used

Abstract

The challenge consisted in elaborate sketches for a website of low and high fidelity, that is, to draw on paper and later pass to the Adobe Illustrator tool, in order to "come to life" and be able to test with possible users the Invision tool was used.

So, the theme chosen was the special effects makeup, so the application intended to give visibility to artists of this world, and at the same time teach some "tricks" to its users, giving the possibility of purchasing products to make this type of makeup. As such, the name chosen for the website was "Mythic Makeup". It was a pair work, and they chose to design for computer, because it made more sense for the theme.
